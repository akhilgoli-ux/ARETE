# Series Mode

A series is a set of connected posts released over consecutive days where each post stands
alone but rewards followers who watch the whole arc. Series build return viewers and give
people a reason to follow rather than just watch once.

## Command
```
/social-ideas series "[series concept]" --parts N --platform [instagram/tiktok/both] --format [format]
```

Examples:
```
/social-ideas series "the 7 stars in your constellation" --parts 7 --platform tiktok --format quote-drop
/social-ideas series "30 days of tracking everything" --platform both --parts 5 --format day-in-my-life
/social-ideas series "what ARETE taught me" --parts 4 --platform instagram --format carousel
```

## How Agent 1 builds a series

### Step 1 — Define the arc
Before writing any individual post, map the emotional journey:

```
SERIES ARC:
  Part 1: PROBLEM     — name the pain, make them feel seen
  Part 2: AGITATION   — deepen the stakes, why this matters more than they think
  Part 3: TURN        — the shift in perspective or the discovery
  Part 4: SOLUTION    — ARETE enters as the answer, not as a product
  Part 5+: PROOF      — what life looks like with the solution in place
  Final:  CALL        — invitation to act (join the waitlist)
```

For longer series (6-7 parts), expand PROOF into multiple posts showing different pillars.

### Step 2 — Write each post as standalone + connected

Every post must:
- Work completely on its own for someone who missed the others
- Contain a forward reference: "Tomorrow: [what's next]" or "Part 2 drops tomorrow"
- Contain a backward thread for returning viewers: a subtle callback to the previous post

### Step 3 — Naming convention
Each post in the series gets a consistent title format:

```
Option A: "Part 1 of 7: [title]"
Option B: "Day 1: [title]" (for daily series)
Option C: Thematic titles that clearly belong together without numbering
```

Use Option C for ARETE — numbered parts feel mechanical. Thematic titles feel like a collection.

Example for "The 7 Stars" series:
- "The star you forgot about" (fitness)
- "The star going dark" (family)
- "The star you keep promising" (goals)
- "The star that feeds all the others" (sleep)
- etc.

### Step 4 — Output format for series

For each part, produce the full post output (hooks, copy, design, hashtags) PLUS:

```
SERIES METADATA:
  Series: [series name]
  Part: [N] of [total]
  Release day: Day [N] (post consecutively)
  Connects to previous: [how this references the last post]
  Forward teaser: [exact text for end-of-post teaser]
  Standalone rating: [1-10 — how well does this work without context?]
```

Every part must score 7+ on standalone rating. If it doesn't, rewrite until it does.

## Series design consistency

All posts in a series share:
- Same background treatment
- Same font system
- Same accent color
- Same brand tag position
- Same series identifier element (a small consistent mark — e.g., a star count ✦✦✦)

The visual consistency is what tells a viewer "this is part of something."

## Posting schedule recommendation

Output a posting schedule at the end of every series:

```
POSTING SCHEDULE:
  Day 1 (Monday):    Part 1 — [title] — [platform]
  Day 2 (Tuesday):   Part 2 — [title] — [platform]
  ...
  Note: Post between 6-8pm local time on weekdays, 10am-12pm on weekends
        (see best-time-to-post.md for platform-specific windows)
```
